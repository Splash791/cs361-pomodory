Pomodory is a basic web application that implements the **Pomodoro Technique** to help users stay focused and boost productivity.

---

### ‚ú® What It Does

This app manages your work and break time through an automatic cycle:

* **Focus Session:** 25 minutes of dedicated work.
* **Short Break:** 5 minutes of rest immediately following a Focus session.

---

### üíª Features

This MVP includes the core functional requirements for a Pomodoro timer:

* **Timer Controls:** Use **Start, Pause, and Reset** buttons to manage your session.
* **Automatic Cycle:** The timer automatically switches between **Focus** and **Break** modes.
* **Save Progress:** Ability to **Save** incomplete Focus sessions to your history.
* **Multi-Page Navigation:** Easily move between the **Timer**, **History**, and **Account** screens.
* **Dark Mode:** A simple toggle is available in the Account screen.
